The IPL Auction Dashboard is a data analytics project designed to analyze and visualize player auction data from the Indian Premier League (IPL). The dashboard helps understand team-wise spending, player demand, base price vs final price trends, and overall auction insights in a clear and interactive way.

This project uses automated data updates through Google Sheets and Google Apps Script, ensuring the dashboard always works with the latest data. The processed data is then used for analysis and visualization to support better decision-making and performance comparison across teams and players.

The goal of this project is to demonstrate real-world data analytics skills such as data cleaning, automation, analysis, and dashboard creation using industry-relevant tools.
